### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/Zlvsky/One-place.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. Configure your .env file
   ```js
    DB_HOST='127.0.0.1'
    DB_USER=''
    DB_PASSWORD=''
    DATABASE=''
   ```
4. Configure your MySQL Database
